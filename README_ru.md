# Страница выпуска клиента Survivalcraft 2.4 NET для многопользовательской игры

[中文](README.md) | [English](README_en.md)

## Введение в проект

Этот проект направлен на предоставление полной функциональности многопользовательской игры для Survivalcraft NET, позволяя игрокам играть, строить и исследовать вместе в одном мире. Проект разработан на C#, поддерживает несколько платформ (Windows, Android, Linux, планируется MacOS, iOS).

### Обновления проекта и обратная связь

#### Обновления проекта
- Мы регулярно обновляем проект, исправляем ошибки и оптимизируем функциональность
- Информацию о последней версии можно найти на главной странице проекта
- Важные обновления выпускаются через Git Release
- Рекомендуем игрокам регулярно проверять обновления для лучшего игрового опыта

#### Отзыв об ошибках
1. Если вы обнаружили ошибку, сообщите об этом, создав Issue с тегом bug, описав проявление ошибки и шаги для её воспроизведения
2. Для предложений о функциональности отправьте Issue с тегом feature

### Способы подключения клиента
1. **Подключение по локальной сети**: Найдите серверы в локальной сети или введите IP-адрес для подключения
2. **Виртуальная локальная сеть**: Используйте стороннее программное обеспечение (например, Sunflower, Hamachi) для создания виртуальной локальной сети и подключения

## Примечания

- Обратите внимание на кроссплатформенную совместимость при игре на разных платформах
- Производительность сервера влияет на максимальное количество поддерживаемых игроков
- Качество сети значительно влияет на игровой опыт; рекомендуется использовать стабильное сетевое соединение
